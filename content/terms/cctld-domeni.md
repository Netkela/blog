**ccTLD (Country Code TLD)** — аббревиатура от «Country Code Top-Level Domain». Расшифровывается, как «Домен верхнего уровня с кодом страны». Значение — национальный домен верхнего уровня, используемый для определения государства либо географического региона/района.

Адрес, который вводится в адресную строку браузера для посещения web сайтов, называется доменом. Часть «после точки» — расширение домена, которое имеет название TLD (Top-Level Domain). Существует несколько типов доменов верхнего уровня, однако в нашем материале поговорим о ccTLD — что представляет собой, о его внедрении и какими преимуществами обладает.

Любая страна, суверенное государство, географическая область либо зависимая территория имеет свой двухбуквенный идентификатор, обозначение — так называемый код, определяющий ccTLD. Некоторые из наиболее распространённых ccTLD:
- **.us** — Соединённые Штаты;
- **.ca** — Канада;
- **.uk** — Соединённое Королевство;
- **.in** — Индия;
- **.au** — Австралия.

Однако многие компании часто предпочитают не использовать ccTLD своего государства, а пользуются доменным именем верхнего уровня, к примеру, .com, .net либо .org. Фактически, ccTLD можно назвать двух буквенным расширением домена, назначающееся государству, стране, штату либо территории на основе ISO.
## Коды стран ISO

International Organization for Standardization (ISO) — Международная организация по стандартизации, занимающаяся с 1947г таким видом деятельности, как разработка коммерческих и промышленных стандартов. Коды стран ISO представляют собой стандартизованные международно признанные средства идентификации государств и стран с применением двух или трёх буквенной комбинации — известной, как ISO alpha-2 и ISO alpha-3. Это как аббревиатура, обозначающая страну или регион — такой подход к определению государств устраняет путаницу, которая может возникнуть из-за различных названий и используемых языков.

Для создания ccTLD используется ISO 3166-1 alpha-2 — стандарт, определяющий двух буквенные коды названий стран, зависимых территорий и особых географических областей. Целью стандарта ISO 3166 является определение международно признанных кодов — букв/цифр того или иного государства.
## Внедрение ccTLD

Country Code Top-Level Domain необходим для использования в целях обслуживания определённого сообщества суверенного государства, конкретной страны, либо автономной республики. ccTLD всегда состоят из двух буквенного кода, присваиваемого каждому государству в соответствии со стандартом ISO-3166 1974г.

Впервые внедрение ccTLD было произведено IANA (Internet Assigned Numbers Authority) — Управлением адресного пространства в сети Интернет, спецподразделением ICANN, ответственным за ведение реестров уникальных идентификаторов в сети Internet, куда включены доменные имена, параметры протокола, адреса IP и номера автономных систем в сети Internet. Управление адресного пространства, которая является руководящим органом, поддерживающим технические стандарты в сети Internet, предоставляет список доступных ccTLD.

Заявки на получение ccTLD впервые стали подаваться в 1985г — первыми, кто зарегистрировался в этом же году, были .us (США), .uk (Великобритания) и .il (Израиль). В настоящее время существует более 300 зарегистрированных ccTLD, среди которых наиболее популярны .cn (Китай), .tk (Tokelau), .de (Германия), .uk (Великобритания), имеющие максимальное количество оформленных доменов. На конец ноября 2021 года официально зарегистрировано более 157,3 млн доменов с ccTLD.
## Преимущественные особенности получения ccTLD 

ccTLD, как домены верхнего уровня с кодом страны, являются важными элементами DNS. Некоторые ccTLD входят в число наиболее часто регистрируемых расширений доменов в мире. С появлением новых общих доменных расширений, ccTLD зачастую становится второстепенным выбором, несмотря на их многочисленные преимущества.

Рассмотрим преимущественные особенности получения Country Code Top-Level Domain:

1. Широкий выбор доменов. Существует около 148 миллионов зарегистрированных доменных имён .com. Это означает, что трудно найти доступные варианты доменных имён. Для сравнения — в большинстве ccTLD зафиксировано менее миллиона регистраций. Следовательно, в этих доменах доступно гораздо больше имён, чем в .com. Больше шансов подобрать доменное имя, если отдать предпочтение ccTLD.

2. Коротко и просто. Country Code Top-Level Domain состоят из двух букв, что делает их самыми короткими доменными расширениями на рынке. Это может стать большим преимуществом, особенно если название компании длинное и нет желания связывать с ним другой TLD. Сочетание короткого домена слева от точки с двух символьным ccTLD позволяет владельцам web сайтов получить короткое и простое доменное имя, запоминающееся клиентами.

3. Доверие потенциальных клиентов. ccTLD можно назвать национальной онлайн идентификацией, которая даёт потенциальным посетителям web сайта больше уверенности. Большинство людей предпочитают пользоваться ресурсами на своём родном языке с местным ccTLD — например, для совершения online покупок и транзакций в региональной валюте.

4. Локальный таргетинг и SEO. При использовании Country Code Top-Level Domain, уже осведомлённый Google ориентируется на конкретное государство для определения наиболее подходящей целевой аудитории для той или иной страницы ресурса. Например, необходимо настроить таргетинг посетителей из Великобритании на интернет-магазин британской эксклюзивной парфюмерии. Можно рассмотреть возможность выбора .co.uk в качестве расширения домена — это гарантирует, что интернет-магазин британской эксклюзивной парфюмерии будете отображаться в результатах поиска тех, кто живёт в Великобритании, а не в Канаде, Австралии, либо другом государстве.

5. Хитрости с названием бренда. Некоторые компании используют ловкий трюк для создания мощного бренда вокруг своего доменного имени — ccTLD позволяет проявить творческий потенциал. Название бренда формируется с использованием доменного имени в сочетании с Country Code Top-Level Domain: часть названия находится слева от точки, а другая часть — справа от точки, таким образом, можно сформировать слово или бренд. Например:

- LinkedIn использует ccTLD Индии .in для создания названия Linked.in;
- YouTube применил бельгийский домен .be в YouTu.be;
- Instagram использует армянский домен .am для создания Instagr.am.

Некоторые ccTLD можно использовать в качестве определённого действия, например: Черногорский домен .me отлично подходит для таких доменов, как love.me и visit.me. Домен США .us можно заменить .us, что в итоге получается love.us или visit.us.

## Заключение

На регистрацию домена верхнего уровня с кодом страны Country Code Top-Level Domain распространяются некоторые ограничения, которые контролируются государственными органами. В конечном итоге правительство может устанавливать свои собственные правила использования домена верхнего уровня с кодом страны ccTLD и лиц, имеющих право их регистрировать. Сюда включены правила и условия, связанные с цензурой и свободой слова. Контролирующие органы некоторых государств требуют проверку и идентификацию личности, прежде чем разрешить регистрацию домена с ccTLD.

Учитывая все обстоятельства, домен верхнего уровня с кодом страны ccTLD может стать отличным выбором для многих компаний, частных лиц и организаций.
