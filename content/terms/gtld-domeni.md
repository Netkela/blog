**gTLD (Generic Top-Level Domain)** — это домен верхнего уровня. Он не привязан к определённому географическому местоположению, как ccTLD и может быть зарегистрирован кем угодно и из любой точки мира. Состоит из трех и более символов. Например .com, .org, .net.
## История создания

Первая волна доменов Generic Top-Level Domain выпущена в 1980г, вскоре после распространения сети Internet. gTLD был разработан, чтобы помочь первому поколению пользователей Интернета организовать свои web сайты. Хотя и домены такого типа уже существуют более 35 лет, первоначальные шесть gTLD являются одними из самых популярных:

- .com — в основном для коммерческих организаций, теперь без ограничений;
- .net — первоначально для сетевых инфраструктур, теперь без ограничений;
- .org — изначально для организаций, теперь без ограничений;
- .xyz — общее назначение;
- .biz — для бизнеса;
- .info — для информационных сайтов.

gTLD без ограничений доступны для регистрации абсолютно любым предприятиям, фирмам и физическим лицам.
## Общие преимущественные характеристики gTLD

Без правильного типа URL-адреса клиенты могут не заинтересоваться услугами или продуктами вашей компании, а ещё хуже — могут запутаться в разных названиях фирм. Для бизнеса не хватало подходящих доменных имен, именно поэтому были созданы общие домены верхнего уровня gTLD. Рассмотрим общие преимущественные характеристики gTLD:

1. Брендинг имеет решающее значение для присутствия в сети Internet, и именно gTLD значительно облегчают компаниям поиск своего уникального бренда, так как они популярны среди пользователей и знакомы всем.
2. Улучшение позиционирования в поисковых системах. Хоть и поисковые системы заявляют, что они не дают преимущества каким-то определенным доменным зонам, но многие специалисты на практике замечали, что это не так. Наиболее популярные gTLD продвигаются легче.
3. Generic Top-Level Domain облегчает распространение web сайта за пределами сети Internet, сокращая потенциальные инвестиции в рекламные и маркетинговые кампании.
## Три лучших домена типа gTLD

В сети Internet доступно более 1000 gTLD, многие из них намекают на функциональность web ресурса: например, «.coffee», «.travel» и др. Их некоторые называют nTLD (new Top-Level Domain). Однако существуют три Generic Top-Level Domain с наибольшим объёмом регистрации — это:

- .com;
- .net;
- .org;

Каждый из этих Generic Top-Level Domain предлагает владельцам доменных имён уникальные преимущества, которые можно использовать наиболее эффективно.
### .com

Общий домен верхнего уровня .com остаётся популярным с момента запуска первой волны Generic Top-Level Domain, имеет такие характерные особенности:

- дословно означает «коммерческий»;
- широко используемый gTLD из когда-либо существовавших;
- самый популярный домен за всю историю.

Если есть планы зарегистрировать доменное имя с использованием .com, тогда следует серьёзно подумать об использовании такой возможности. В преимущества пользованием .com входит:

- Высокое доверие. Пользователи сети Internet склонны доверять web сайтам «.com».
- SEO. Сайты с доменным именем «.com» имеют больше шансов оказаться в верхней части результатов поиска.
### .net

Лидерство .com не обсуждается, однако есть gTLD, практически не уступающий в объёме зафиксированных регистраций. Он имеет такие характерные особенности:

- дословно означает «сеть»;
- существует с 1980-х годов;
- один из самых доступных.

Значение «.net» указывает на то, что этот Generic Top-Level Domain .net изначально предназначался для технологических компаний и отраслей. Он часто используется для web ресурсов, продающие, рекламирующие и продвигающие услуги в сети. Поскольку зарегистрировано меньше доменных имен «.net», нежели доменных имен «.com», у частных лиц либо компаний имеется больше шансов получения доменного имени «.net», наиболее подходящих их бренду.
### .org

.org — ещё один общий домен верхнего уровня, доступный для регистрации любому пользователю. Однако изначально он предназначался для обозначения порталов некоммерческих и неправительственных организаций. Подобно «.net» и «.com» «.org» является одним из старейших, наиболее надёжных и доступных gTLD. Характерные особенности:

- дословное значение — сокращённо «организация»;
- порталы «.org» часто ориентированы на создание сообществ.

Существуют миллионы вариантов доменных имен .org, поэтому больше шансов получить желаемое доменное имя с таким gTLD. В преимущества пользованием .org входит:

1. Хороший выбор для SEO. В поисковых системах работает достаточно хорошо, поскольку находится на втором уровне предпочтительных gTLD наряду с .net.
2. Заслуживают доверия пользователей. Самые популярные ресурсы «.org», например, Википедия, обладают повышенным уровнем доверия, предоставляют достоверную информацию, функционируют как сообщество единомышленников.
### Открытие новых gTLD

Корпорацией ICANN постоянно предоставляет доступ к новым доменным зонам: «.aero», «.biz», «.coop», «.info», «.museum», «.name» и «.pro» — это первые «новые gTLD» (nTLD).

В течение следующих нескольких лет корпорация ICANN планирует не останавливаться и расширять доменные зоны для компаний, чтобы каждый смог найти себе короткое имя.
