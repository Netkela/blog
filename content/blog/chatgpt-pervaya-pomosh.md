Купил плагин на WordPress для перевода статей, в нем оказалось старое API Yandex Translate 1,5. Написал разработчику-индусу с просьбой обновить API версию в плагине до 2.0, так как старая уже не поддерживается Яндексом.

Он мне ответил, что это первый запрос на такую доработку за все время и раньше Yandex Translate 1,5 был бесплатен, поэтому он его реализовал. В общем, дал понять, что обновлять не будет.

Разговорился сегодня со знакомым, он с помощью ChatGPT написал свой движок для сайта. Думаю, попробую и я дать команду переделать версию API.

Скормил ChatGPT файл от этого WordPress плагина, попросил переписать код под новую версию API 2.0. Заменил код в файле плагина, который он мне выдал, и все заработало.

При этом, я вообще не понимаю в программирование. Делов на пару минут и новая рабочая версия плагина готова, ну это ли не магия.
